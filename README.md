# node-express-pg-graphql-template

## Getting Started

Run the development server:

Terminal 1:

```bash
npm run watch
# or
yarn watch
```

Terminal 2:

```bash
npm run dev
# or
yarn dev
```

Run development worker:

Terminal 3:

```bash
npm run worker-dev
# or
yarn worker-dev

Open [http://localhost:8000/graphql](http://localhost:8000/graphql) with your browser.
```
